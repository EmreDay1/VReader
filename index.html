VimDoc Reader - Command Reference
Navigate with j/k, press q or Esc to close
Basic Navigation
h, j, k, l Move left, down, up, right
w, b, e Jump to word start/end
gg, G Go to document start/end
:n Jump to page n
0, $, ^ Go to line start/end/first char
Screen Movement
Ctrl+f, Ctrl+b Page down/up
Ctrl+d, Ctrl+u Half page down/up
zz, zt, zb Center/top/bottom viewport
gj, gk Move by visual line (wrapped text)
Document Structure Navigation
]] Next section/heading
[[ Previous section/heading
]} Next paragraph
[{ Previous paragraph
Search & Find
/pattern Search forward for pattern
?pattern Search backward for pattern
n, N Next/previous search match
*, # Search for word under cursor
:noh Clear search highlighting
f{char}, F{char} Find next/prev occurrence in line
t{char}, T{char} Go to before/after char in line
;, , Repeat last find command
Bookmarks & History
m{a-z} Set bookmark labeled a-z
'a Jump to bookmark a
:marks List all bookmarks
Ctrl+o, Ctrl+i Go back/forward in jump list
g;, g, Go to position of last/newer edit
View Controls
:set linebreak Toggle line wrapping at word boundaries
:set wrap/nowrap Toggle line wrapping
:set nu/nonu Toggle line numbers
+, -, = Increase/decrease/reset font size
Document Management
:w Save annotations/notes
:e filename Open document
:q, :qa Close document/all documents
:wq Save and close
:ls List all open documents
